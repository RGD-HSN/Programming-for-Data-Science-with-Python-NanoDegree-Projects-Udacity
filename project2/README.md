# US Bikeshare Data Analysis

## Project Overview
This project explores bikeshare data from three major US cities: **Chicago, New York City, and Washington**.  
It allows users to interactively filter the data by city, month, and day, and then computes various statistics to uncover usage patterns.

The goal of this project is to practice Python programming, data analysis using Pandas, and exploratory data analysis skills.

---

## Datasets Used
The program analyzes data from the following CSV files:

- `chicago.csv`
- `new_york_city.csv`
- `washington.csv`
The datasets are provided by Udacity and are not included in this repository due to size limitations:


Each dataset contains information about individual bike trips, including:
- Start and end time
- Start and end stations
- Trip duration
- User type
- Gender and birth year (for some cities)

---

## Features

The program allows users to:

- Filter data by **city**
- Filter data by **month**
- Filter data by **day of the week**
- View descriptive statistics including:
  - Most frequent times of travel
  - Most popular stations
  - Trip duration statistics
  - User type breakdown

## 🚀 How to Run the Project

### 1. Install dependencies
```bash
pip install pandas numpy
```

### 2. Run the script
python bikeshare.py

### 3. Follow the prompts

The program will ask you to choose:

City
Month
Day

Then it will display computed statistics.
