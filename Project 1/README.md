# DVD Rental SQL Analysis Project

##  Overview

This project is part of the Programming with Data Science for Python Nanodegree. It focuses on analyzing a DVD rental database using SQL to extract meaningful insights about customer behavior, movie categories, and store performance.

The goal of this project is to demonstrate SQL skills, including joins, aggregations, window functions, and filtering techniques to answer real-world business questions.

---

## Objectives

* Analyze family-friendly movie categories
* Understand rental patterns across stores and time
* Identify top customers and their payment behavior
* Practice writing efficient and structured SQL queries
---

## SQL Techniques Used

* JOIN (multiple tables)
* GROUP BY and aggregation functions (COUNT, SUM)
* Window functions (NTILE)
* Date functions (DATE_PART, DATE_TRUNC, TO_CHAR)
* Common Table Expressions (CTEs)

---

## 🚀 Conclusion

This project demonstrates how SQL can be used to transform raw relational data into meaningful insights. It highlights the importance of structured querying in understanding business performance and customer behavior.
