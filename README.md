## Projects Included

### 1. SQL Data Exploration & Analysis

This project focuses on writing SQL queries to analyze a relational database.

Key tasks include:

* Analyzing family-friendly movie categories and rental frequency
* Segmenting films based on rental duration using quartiles
* Comparing store performance over time
* Identifying top customers and analyzing their payment behavior

The project includes:

* SQL query file
* Resulting dataset (CSV)
* Presentation file (ODF) summarizing key findings
